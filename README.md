# Digital Nurture Java FSE Hands-On Work

This repository holds my weekly exercise implementations and assignments for the Digital Nurture Java Full Stack Engineer (FSE) program. All code has been written cleanly and is comment-free.

## What is inside:

### 📂 week1
*   **Web Portal Basics:** Basic HTML5 structure, CSS3 styling, and JavaScript logic for the event portal.
*   **Bootstrap 5:** A clean, responsive template of the portal.
*   **ANSI SQL:** Database tables, test insertions, and solutions for the 25 MySQL query exercises.
*   **Core Java:** 41 programs ranging from simple calculators and conditionals to collections, threading, streams, and JDBC.

### 📂 week2
*   **Design Patterns:** Java implementations of 11 GoF design patterns (Singleton, Builder, Strategy, Observer, MVC, etc.).
*   **Algorithms & DSA:** Core search and sorting tasks, list structures, and forecasting recursion.
*   **PL/SQL:** Stored procedures, triggers, cursors, and custom packages.
*   **Java Testing & Logging:** Maven project with JUnit 5 tests, Mockito stubs, and SLF4J logging configuration.
*   **Spring & JPA:** Maven projects showcasing XML context, DI injection, Aspects, JPA databases, and REST APIs.
*   **Microservices:** Multi-module Maven setup featuring a Eureka Server, API Gateway, and Account/Loan services.
*   **React:** A dashboard component testing and displaying the 19 React Labs.
*   **Angular:** Workspace template for the Student Course Portal.
*   **Git:** Commands log sheet for basic repository versioning.
